How to install in Gmail

  1. Click Copy signature below
  2. Open Gmail → Settings (gear icon) → See all settings
  3. Go to the General tab → scroll to Signature
  4. Click Create new, give it a name (e.g. "ShotSync")
  5. Click inside the signature text box and paste ⌘V
  6. Scroll down and click Save Changes
